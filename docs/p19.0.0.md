>
>```shell
>import yaml
>
>Python = yaml.loads(YAML)
>Python = yaml.load(File)
>
># Python => YAML 
>YAML = yaml.dumps(Python, ensure_ascii = False)
>YAML = yaml.dump(Python, File, ensure_ascii = False, encoding = "UTF-8")
>
>
>```
>
>
