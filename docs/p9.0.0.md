<br/>

!> 需要导入模块 [`import random`]

```shell
# 生成0到1之间的浮点数
random_float = random.random()
print(random_float)
 
# 生成5到10之间的浮点数
random_uniform = random.uniform(5, 10)
print(random_uniform)
 
# 生成1到10之间的整数
random_int = random.randint(1, 10)
print(random_int)
 
# 生成1到10之间的整数，步长为2
random_range = random.randrange(1, 10, 2)
print(random_range)
 
# 从列表中随机选择一个元素
random_choice = random.choice([1, 'a', 3.14, 'hello'])
print(random_choice)
 
# 从列表中随机抽取两个元素
random_sample = random.sample([1, 'a', 3.14, 'hello'], 2)
print(random_sample)
 
# 将列表中的元素随机排序
lst = [1, 2, 3, 4, 5]
random.shuffle(lst)
print(lst)
    
    
```
