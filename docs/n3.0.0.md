!> 在`Python`中，用于处理与时间相关的操作 [`import time`]

```csharp
import time

{

      #region 常用函数

      // [ 暂停 ] 程序执行指定的秒数
      time.sleep(seconds)

      /// <summary>
      /// - 高精度计时器的值 ( 用于测量短时间间隔 )
      ///   start = time.perf_counter()
      ///   time.sleep(1)
      ///   end = time.perf_counter()
      ///   print(end - start)
      /// 
      /// </summary>
      time.perf_counter()

}


```
