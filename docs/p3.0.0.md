#### **[ 变量 ]数据类型**

```shell
# 使用 type( ... ) 查看变量数据类型
_v1 = 50
_v2 = 50.0
_v3 = "Hello"

# 数据类型： <class 'int'> <class 'float'> <class 'str'>
v1_type = type(_v1)
print("数据类型：", v1_type, type(_v2), type(_v3))
    

```

####  **[ ☢ 变量 ]类型转换**

>[!WARNING|style: flat|label: 简要说明 ]
>
>- `int(x)`：将`x`转换为一个整数
>- `float(x)`：将`x`转换为一个浮点数
>- `str(x)`: 将对象`x`转换为字符串
>
><br/>

```shell
v1 = "10"
int_v1 = int(v1)

v2 = 10.52
int_v2 = int(v2)  #[ 注意 ]丢失精度

# 10 10
print(int_v1, int_v2)


```
