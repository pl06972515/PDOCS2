>- [`input( Message )`] 数据输入 <span style='color:red'>[ 等待键盘输入 - 接收内容永远是`string`]</span>
>- [`print( p1, p2, p3 ... )`] 数据输出
>
><br/>

```shell
print("请输入姓名：")
_name = input()
_age = int(input("请输入年龄: "))

print(f"姓名： { _name }, 年龄： { _age }")
    
    
```
