>[!WARNING|style: flat|label: 简要说明 ]
>
>- <span style='color:red'>(`threading.RLock`) 最基本的锁机制，用于确保同一时间只有一个线程可以执行某个代码块 [`import threading`]</span>
>
>  <span style='color:red'>[ 通过`with lock:`自动处理锁的获取和释放 ] [ 注意事项：锁支持重入 ]</span>
>
>```csharp
>from threading import RLock
>
>_lock = RLock()
>{
>
>      # _lock.acquire(blocking = True, timeout = -1):
>      #   - blocking = True  阻塞请求: 直到锁变得可用 | False 非阻塞锁: 如果锁当前被其他线程持有则返回 False 而不会阻塞
>      #   - timeout > 0: 最多等待 N 秒
>
>      _lock.acquire(blocking)
>      _lock.release()
>
>}
>
>
>```
>
>
>
><br/>
>
>

```csharp
_lock = RLock()
def M1():
    # [ 阻塞锁 ] with: 自动 .acquire() → .release()
	with _lock:
		print('逻辑处理: 同一时刻只允许一个线程处理!')
            
            
```

```csharp
_lock = RLock()
def M1():

	# [ 尝试获取锁 ]
	if not _lock.acquire(blocking = False):
		return None

	try:
		print(f"逻辑处理: 执行实际业务逻辑 !")
	except Exception as e:
		print(f"其他错误: {e}")
	finally:
		# 确保释放锁
		_lock.release()
	return None
            
            
```
