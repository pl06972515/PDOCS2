!> （`Python`) 使用`lambda`关键字，来创建匿名函数 <span style='color:red'>[ 匿名函数：不需要使用`def`关键字定义 ]</span>

```csharp
// Action
M1 = lambda: print("Hello, world!")
M1()

// Action<int, int>
M1 = lambda x, y=30: print(x + y)
M1(10)
    
    
```

```csharp
// Func<int>
M1 = lambda : 100
x = M1()

// Func<int, int, int>
M1 = lambda x, y=30: x + y
x = M1(10)
    
    
```
